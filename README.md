Giới thiệu
----------------------------------------------
LaptopShop là một website thương mại điện tử cho phép người dùng xem, tìm kiếm và mua laptop trực tuyến. Dự án được xây dựng với kiến trúc rõ ràng, tách biệt frontend – backend – database, giúp dễ dàng mở rộng và bảo trì.

Website bao gồm các chức năng quản lý sản phẩm, người dùng, giỏ hàng, đơn hàng… thích hợp để học tập, thực hành hoặc phát triển thành một website bán hàng thực tế.

Tính năng chính 

- Tính năng người dùng
  + Xem danh sách laptop kèm mô tả, cấu hình, giá và hình ảnh
  + Tìm kiếm và lọc theo thương hiệu, danh mục, giá
  + Thêm sản phẩm vào giỏ hàng
  + Đặt hàng và xem lịch sử mua hàng
  + Đăng ký, đăng nhập tài khoản

- Tính năng quản trị (Admin)
  + Quản lý sản phẩm laptop: thêm, sửa, xoá
  + Quản lý danh mục / thương hiệu
  + Quản lý người dùng
  + Quản lý đơn hàng
    
- Tính năng khác
  + Giao diện responsive
  + Hệ thống đăng nhập bảo mật
  + Kết nối cơ sở dữ liệu MySQL
  + Cấu trúc dự án theo mô hình MVC

Công nghệ sử dụng
- Backend
  + Spring Boot
  + Spring MVC
  + Spring Data JPA / Hibernate
  + JSTL, JSP cho giao diện phía server

- Frontend
  + HTML5, CSS3, JavaScript
  + Bootstrap 

- Cơ sở dữ liệu
  + MySQL
